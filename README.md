# IES_Project

Application for gathering entry/exit data of people in a building.

## Data interaction

There are 3 different interfaces:
- **Student**
  - username: Student
  - password: Password
- **Security Guard**
  - username: Security
  - password: Password
- **Analyst**
  - username: Analyst
  - password: Password
